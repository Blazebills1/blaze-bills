Blaze Bills

This is a simple iPhone web app for bill tracking.

Best iPhone setup:
1. Upload index.html to a free host like Netlify Drop or GitHub Pages.
2. Open the hosted link in Safari.
3. Tap Share.
4. Tap Add to Home Screen.
5. Name it Blaze Bills.

The app stores data locally on your iPhone using browser storage.
Use Settings > Export Backup to save a copy of your data.
